# analog2PWM
 Arduino Nano zur PWM Steuerung von PC Lüftern

Mein Computer Mainboard hat überwiegend nur analog gesteuerte Lüfter verbaut die mir zu laut waren.
Um leisere PWM gesteuerte Lüfter einsetzen zu können habe ich dieses kleine Programm
für einen Arduino Nano geschrieben der die analoge Spannung für die Lüfter einzeln
abgreift und dann in PWM Signale umsetzt.
